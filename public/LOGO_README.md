# Logo Replacement Instructions

## Klinikum Dortmund Logo

The application is configured to use the Klinikum Dortmund logo in the header toolbar.

### To replace the placeholder logo:

1. Download the official logo from:
   ```
   https://upload.wikimedia.org/wikipedia/commons/thumb/0/0e/Klinikum_Dortmund_Logo.svg/320px-Klinikum_Dortmund_Logo.svg.png
   ```

2. Save it as `klinikum-logo.png` in the `/public` directory, replacing the current placeholder file.

3. The logo dimensions should be 320 Ã— 74 pixels for optimal display.

4. Rebuild the application:
   ```bash
   pnpm run build
   ```
